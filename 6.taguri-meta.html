Metatags specifica metadata si stau in tagurile head din fisierul HTML. Elementele meta pot folosi perechi de nume/valoare, pt a descrie proprietatile documentului HTML: autor, data, lista de cuvinte cheie. Tagul meta e self enclosed, aduce info suplimentare.

  • Taguri pt specificitate

    - specifica cuvinte cheie, relavnte pt documemnt. Cuvinele sunt folosit ede motoare de cautare, pt indexarea paginii - faciliteaza cautarea
    • <meta name="keywords" content="HTML, Meta Tags, Metadata" />

  • Taguri pentru descrierea documentului

    - ofera o descriere scurta a documentului. POate fi filosit de motoarele de cautare pt indexare
    • <meta name="keywords" content="HTML, Meta Tags, Metadata" />
      <meta name="description" content="Learning about Meta Tags." />

  • Taguri pt info referitoare la rvizia doc

    - cand a fost revizuit ultima data documentul? Pot fi folosite la refresh
    • <meta name="keywords" content="HTML, Meta Tags, Metadata" />
      <meta name="description" content="Learning about Meta Tags." />
      <meta name="revised" content="MDN, 30/06/2021" />

  • Refresh la document

    - Pt ca pagina sa-si dea refresh dupa 5secunde:
    • <meta name="keywords" content="HTML, Meta Tags, Metadata" />
      <meta name="description" content="Learning about Meta Tags." />
      <meta name="revised" content="MDN, 30/06/2021" />
      <meta http-equiv = "refresh" content = "5" />

  • Redirectionarea paginii

    redirectionare catre alta pagina dupa 5 secunde
    • <meta http-equiv = "refresh" content = "5; url = http://google.com" />

  • Setare cookies

    Cookies - date stocate in fisiere text in computer, si sunt schimbate intre browser si serverul web. In exempu, daca nu este inclusa data expirarii - cookie sunt considerate cookie de tip session si vor vi sterse cand userul paraseste browserul
    • <meta http-equiv = "cookie" content = "userid = adeer; expires = Wednesday, 08-Aug-15 23:59:59 GMT;" />

  • Setare nume de autor

    • <meta name = "author" content = "Adina" />

  • Specificare tipuri de caractere

    By default, browserele web folosesc procesul de codare ISO-8859-1(Latin1). Exemplu de codare UTF-8
    • <meta http-equiv = "Content-Type" content = "text/html; charset = UTF-8" />
