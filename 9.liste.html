Tipuri de liste:

  • Neordonate -> ul
  • Ordonate -> ol
  • Definitie -> dl
Liste Ordonate:

Atributul type pentru liste ordonate:

<ol type="1"></ol>

Atributul start arata de unde se porneste numararrea

<ol type="1" start="4">
  • type = "1" -> numere arabe
  • type = "I" -> numere romane: III
  • type = "i" -> numere romane: iii
  • type = "A" -> litere mari: A,B,C..
  • type = "b" -> litere mici: a,b,c..
<ol type = "I" start = "2">
<li>HTML</li>
<li>CSS</li>
<li>JS</li>
</ol>
  1. HTML
  2. CSS
  3. JS
Liste neordonate:

Atributul type in lista neordonata arata de ce fel este bullet pont: square, disc, circle

<ul type="square">
<li>HTML</li>
<li>CSS</li>
<li>JS</li>
</ul>
  • HTML
  • CSS
  • JS
Liste de tip definitii:
  • <dl> -> inceputul listei (definition list)
  • <dt> -> termenul de definit (definition term)
  • <dd> -> definitia termenului (definition)
  • </dl> -> sfarsiyul listei (end of list)
<dl>
<dt><b>HTML</b></dt>
<dd>This stands for Hyper Text Markup Language</dd>
<dt><b>HTTP</b></dt>
<dd>This stands for Hyper Text Transfer Protocol</dd>
</dl>
HTML
This stands for Hyper Text Markup Language
HTTP
This stands for Hyper Text Transfer Protocol
