Cu atributele definim caracteristicile unui element HTML.
Atributele sunt plasate in tagul de deschidere al elementului.
Toate atributele sunt alcatuide din 2 parti: nume si valoare

Numele este proprietate setata. De exemplu, tagul <p> din elementul paragraf are atributul cu denumirea align care indica ranjarea paragrafului in pagina.

Valoarea se seteaza si este mereu intre "". In exemplu sunt descrise 3 valori posibile pt aliniere: left, center si right.

<p align="left"> This is left aligned</p>

This is left aligned

This is center aligned

This is right aligned

Atribute de baza

  • Id - identificare rapida a unui element unic - dupa id
  • Title - afisat ca tooltip (mouseover, hover - display text)

    • Titled Heading Tag Example

  • class - asociere elemet cu stylesheet
    • class = "className1 className2 className3"
  • style - permite atribuire CSS in direct element

    • Text rosu

Internationalization Attributes

  • dir - indica directia de afisare a textului in pagina
    • ltr - left to right
    • rtl - right to left
  • lang - limba principala folosita in document
    • ex pt limba engleza = en
  • xml:lang - alta met de indicare limba in doc (ISO 639 = codul tarii)
