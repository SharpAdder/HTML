Cea mai rapida metoda de a adauga muzica, video in pagina este cu ajutorul tagului embed

<embed src="/html/yourfile.mid" width="100%" height="60">
<noembed><img src="yourimage.gif" alt="Alternative Media"></noembed>
</embed> <img src="yourimage.gif" alt="Alternative Media">
Lista de atribute pentru embed:
  • align - invechit
  • autostart - invechit
  • loop - sunetul va rula continuu
  • playcount - de cate ori va rula un sunet
  • hidden - specifica daca obiectul multimedia va aparea in pagina
  • width - latimea obiectului, in pixeli
  • height - inaltimea obiectului, in pixeli
  • name - nume de referinta pt obiect
  • src - URL-ul obiectului
  • volume - conrollere de volum
HTML Object - tag

Folosit pt includrea obj in pagina

Embeded docuemnt html

<object data="data/test.html" type="text/html" width="300" height="200">
alt : <a href="data/test.html">test.htm</a>
</object>

PDF embeded in HTML

<object data="data/test.pdf" type="application/pdf" width="300" height="200">
alt : <a href="data/test.pdf">test.htm</a>
</object>
