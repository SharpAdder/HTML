Tabelele in HTML sunt create cu tagul table care contine 2 taguri: tr - table row -> creaza randuri in tabel si td - table data -> creaza celulele din tabel

Tabel cu 1 bordura
<table border="1">
 <tr>
  <td>Row 1, Column 1</td>
  <td>Row 1, Column 2</td>
 </tr>

 <tr>
  <td>Row 2, Column 1</td>
  <td>Row 2, Column 2</td>
 </tr>
</table>
Row 1, Column 1 Row 1, Column 2
Row 2, Column 1 Row 2, Column 2
Titluri in tabel - table hradings

Tagul th va fi pus deasupra si va inlocuiu tagul td

<table border="1">
<tr>
<th>Name</th>
<th>Salary</th>
</tr>

<tr>
<td>John Doe</td>
<td>3000$</td>
</tr>

<tr>
<td>Jane Doe</td>
<td>3000$</td>
</tr>
</table>
Name Salary
John Doe 3000$
Jane Doe 3000$
Spatiere intre celule

Atribuemtele cellpaddingsi cellspacing

  • cellpadding - distanta dintre content si bordura
  • cellspacing - distanta intre celule
<table border="1" cellpadding="5" cellspacing="5">
<tr>
<th>Name</th>
<th>Salary</th>
</tr>

<tr>
<td>John Doe</td>
<td>3000</td>
</tr>

<tr>
<td>Jane Doe</td>
<td>3000</td>
</tr>
</table>
Name Salary
John Doe 3000
Jane Doe 3000
Atributele colspan si rowspan
  • colspan - folosit pt extinderea pe mai multe coloane
  • rowspan - folosit pe extinderea pe mai multe randuri
<table border="1">
<tr>
<th>Column 1</th>
<th>Column 2</th>
<th>Column 3</th>
</tr>

<tr>
<td rowspan="2">Row 1 Cell 1</td>
<td>Row 1 Cell 2</td>
<td>Row 1 Cell 3</td>
</tr>

<tr>
<td>Row 2 Cell 2</td>
<td>Row 2 Cell 3</td>
</tr>

<tr>
<td colspan="3">Row 3 Cell 1</td>
</tr>
Column 1 Column 2 Column 3
Row 1 Cell 1 Row 1 Cell 2 Row 1 Cell 3
Row 2 Cell 2 Row 2 Cell 3
Row 3 Cell 1

Descriere tabel - tagul caption invechit
Header, body & footer
  • thead - creaza header separat
  • tbody - indica corpul tabelului
  • tfoot - creaza footer separat
<table border="1" width="100%">
<thead>
<tr>
<td colspan="4">This is the head of the table</td>
</tr>
</thead>

<tfoot>
<tr>
<td colspan="4">This is the foot of the table</td>
</tr>
</tfoot>

<tbody>
<tr>
<td>Cell 1</td>
<td>Cell 2</td>
<td>Cell 3</td>
<td>Cell 4</td>
</tr>
</tbody> </table>
This is the head of the table
This is the foot of the table
Cell 1 Cell 2 Cell 3 Cell 4
