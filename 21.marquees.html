HTML marquee - text care semisca pe orizontala sau verticala. Tagul marquee este invechit, nu se foloseste.Sunt indicate Js si CSS pentru efecte.

Sintaxa:

<marquee attribute_name="attribute_value" ....more attributes>
One or more lines or text message or image </marquee>
Atribute
  • width --> latimea ariei de rulare text
  • heigth --> inaltimea ariei
  • direction --> directia de rulare text
  • behavior --> scroll, slide, alternate
  • scrolldelay --> intarzierea
  • scrollamount
  • loop --> de cate ori va rula textul
  • bgcolor --> culoare fundal
  • hspace --> spatiul orizontal in jurul marquee
  • vspace --> spatiul pe verticala in jurul marquee
Exemple de marquee
<marquee>This is basic example of marquee</marquee>
This is basic example of marquee
<marquee width="50%">This example will take only 50% width</marquee>
This example will take only 50% width
<marquee direction="right">This text will scroll from left to right</marquee> This text will scroll from left to right
<marquee direction="up">This text will scroll from bottom to up</marquee>
This text will scroll from bottom to up
